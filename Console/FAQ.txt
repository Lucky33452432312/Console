[ FAQ ]

Q: Why can't I enter a command?
A: Make sure you have no capital letters, also try checking that you've entered the command name properly by typing "help".

Q: I can't use File Creator
A: Make sure your file name doesn't have a space in it and make sure the file extension exists

Q: Why can't I type "cd .."?
A: This command works differently. Type "cd" and it will say "Select The Directory You Want To CD To:" and then type ..
